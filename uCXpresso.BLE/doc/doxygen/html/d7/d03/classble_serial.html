Detailed Description

bleSerial class is a ble core, and inherits from CStream class to provide the stream virtual functions for serial input and output. the bleSerial class also inherits from the CThread class and can be work in background.

Remarks
1.The bleSerial will use an interrupt resource, so remain 7 interrupts can be used.
2.The bleSerial will become to a Weakup Source for Power Save Features.
Examples:
/ble/ble_heartrate/src/main.cpp, and /ble/ble_multiple/src/main.cpp.

virtual void CThread::run ( )
protectedvirtualinherited

The run member function is the task main procedure, and callback by RTOS when task start.

Example:
class CLedTask: public CThread {
protected:
virtual void run() {
CPin led(LED2);
while( isAlive() ) { // check thread alive
led = !led;
sleep(200);
}
}
};
int main() {
...
CLedTask ledTask;
ledTask.start("led"); // default stack=128, default priority=low
...
}
Remarks
The run() is a pure virtual function, must implement by inheritor.
Note
if end the run() member function, the CThread object will be destroyed and collected.

Reimplemented in CDebug, and CShell.

Enable a watchdog on a BLE connection. The watchdog feature will cause the BLE core reset when remote (App) crash or lose the connection.
