Detailed Description

The CThread class provide the multi-tasking services.

Note
The CThread class is an abstract class, child class have to implement the run() member.
Examples:
/ble/ble_heartrate/src/main.cpp, /misc/garbage_collector/src/main.cpp, /peripheral/uart/src/main.cpp, /peripheral/watchdog/src/main.cpp, and /rtos/blink_thread/src/main.cpp.

virtual void CThread::run ( )
protectedvirtual

The run member function is the task main procedure, and callback by RTOS when task start.

Example:
class CLedTask: public CThread {
protected:
virtual void run() {
CPin led(LED2);
while( isAlive() ) { // check thread alive
led = !led;
sleep(200);
}
}
};
int main() {
...
CLedTask ledTask;
ledTask.start("led"); // default stack=128, default priority=low
...
}
Remarks
The run() is a pure virtual function, must implement by inheritor.
Note
if end the run() member function, the CThread object will be destroyed and collected.

Reimplemented in CDebug, and CShell.

Examples:
/ble/ble_heartrate/src/main.cpp, /misc/garbage_collector/src/main.cpp, /peripheral/uart/src/main.cpp, /peripheral/watchdog/src/main.cpp, and /rtos/blink_thread/src/main.cpp.
