Detailed Description

The Console class provides a lightweight input/output stream to console.

Constructor & Destructor Documentation

Console::Console ( CStream &  io)

Constructs a console object.

Parameters
iois a CStream object to provide the input/output stream.
Example:
CSerial cdc(USB); // create an USB serial stream object.
Console con(cdc); // create a console by the USB stream.
if ( con.isConnected() ) {
con << "Hello World" << endl;
}
