Detailed Description

A digital input/output bus, used for reading/writing the state of a collection of pins.

Examples:
/ble/ble_heartrate/src/main.cpp, /ble/ble_multiple/src/main.cpp, /peripheral/adc/src/main.cpp, /peripheral/eeprom/src/main.cpp, /peripheral/i2c_scanner/src/main.cpp, /peripheral/spi/src/main.cpp, /peripheral/timer/src/main.cpp, and /peripheral/watchdog/src/main.cpp.

Constructs a CBus object to connect to the specified pins.

Parameters

Example:

bus.output(); // Set the bus as output pins.
