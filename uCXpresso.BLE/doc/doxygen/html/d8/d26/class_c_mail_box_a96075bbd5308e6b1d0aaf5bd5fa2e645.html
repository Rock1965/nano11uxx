static bool CMailBox::post ( LPCTSTR  name,
MESSAGE_T  msg,
uint32_t  timeout = 0 
)
static

Call the member function to send a message to receiver which the same mailbox name.

Parameters
nameis a string to point to which the same name of mailbox to be received the message.
msgis a pointer to the message (void*).
timeoutis a time to wait for post. (Default zero)
Returns
true if post the message successful; otherwise, pose failed.
Example:
class Task1: public CThread {
protected:
virtual void run() {
int i=0;
while(1) {
CMailBox::post(MAIL_LED, &i);
sleep(500);
}
}
};
Remarks
The CMailBox::post is a static function.
