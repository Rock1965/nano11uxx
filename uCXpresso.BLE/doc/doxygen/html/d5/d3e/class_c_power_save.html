Detailed Description

CPowerSave provides the low power feature by Tickless Technology.

Remarks
Don't use the CPowerSave and CWatchdog together, because the CPowerSave will use the watchdog as a weakup source.
Examples:
/ble/ble_multiple/src/main.cpp, /ble/ble_proximity/src/main.cpp, and /peripheral/power/src/main.cpp.
