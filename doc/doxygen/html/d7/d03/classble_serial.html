Detailed Description

bleSerial class is a ble core, and inherit from CStream class to provide the stream virtual functions for serial input and output. the bleSerial class also inherit from the CThread class and can work in background.

void bleSerial::watchdog ( uint32_t  tm)

Enable a watchdog on a BLE connection. The watchdog feature will cause the BLE core reset when remote (App) crash or lose the connection.

Parameters
tmA timeout value in millisecond, recommend value is 500~30000. If set the tm to zero, it is meant to disable the watchdog.
Note
The member is an optional function, and default is 10,000ms (10 seconds).
