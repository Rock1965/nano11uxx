virtual void CShell::run ( )
virtual

The run member function is the task main procedure, and callback by RTOS when task start.

Example:
class CLedTask: public CThread {
protected:
virtual void run() {
CPin led(LED2);
while(1) {
led = !led;
sleep(200);
}
}
};
int main() {
...
CLedTask ledTask;
ledTask.start("led"); // default stack=128, default priority=low
...
}
Remarks
The run() is a pure virtual function, must reimplement by inheritor.
Note
if end the run() member function, the CThread object will be deleted and collected.

Reimplemented from CThread.
