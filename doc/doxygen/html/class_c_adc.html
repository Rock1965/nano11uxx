Read the ADC value from the AD pin with Compare-Filter

Parameters
[in]filtera float value to identify the offset range with last value. (0.1 = 10%, 0.25=25%)
[in]counta integer value to identify to retry count if ADC value over the filter range. (default value is 3)
Returns
12 bits resolution (0~4095) ADC value
Example:
CAdc ad(AD0);
ad.begin();
int value = ad.read(0.25, 3); // if AD value over 25% with last value, try to read the AD value 3 times.

Returns
12 bits resolution (0~4095) ADC value
Example:

ad.begin();
CTime t(TIMER0);
t.setting(10, 50); // 10x50 = 500us interval
t.begin();
